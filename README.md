# Child Vaccination Classification Using Machine Learning

## Project Overview

This project applies binary and multiclass machine learning approaches to classify childhood vaccination status using demographic, maternal healthcare, and socioeconomic indicators.

## Objectives

- Predict childhood vaccination status
- Compare binary and multiclass classification approaches
- Evaluate multiple machine learning models
- Identify important predictors of vaccination

## Dataset

- SPSS Dataset (`.sav`)
- Processed Excel Dataset (`.xlsx`)

## Notebooks

### 1. Data Cleaning and Preprocessing
- Data cleaning
- Missing value handling
- Feature engineering
- Vaccination target creation

### 2. Exploratory Data Analysis
- Distribution analysis
- Vaccination coverage analysis
- Correlation analysis
- Visualization

### 3. Binary Machine Learning Modeling and Evaluation
Models:
- Logistic Regression
- Ridge Logistic Regression
- Random Forest
- XGBoost
- KNN
- SVM

### 4. Multiclass Machine Learning Modeling and Evaluation
Models:
- Multinomial Logistic Regression
- Random Forest
- XGBoost
